# 적고추 재배단계별 서브에이전트 수집 체계

이 문서는 적고추(건고추) 재배 지식 확장을 위해 재배단계별 서브에이전트가 어떤 범위를 맡고, 어떤 자료를 우선 수집하며, 어떤 기준으로 RAG seed chunk로 반영하는지 정리한다.

## 목적

- 생육 단계마다 필요한 판단 근거를 분리해 `RAG` 검색 정밀도를 높인다.
- 중복 지식은 줄이고, 단계별로 부족한 실무형 규칙만 추가한다.
- 스마트팜 AI 어시스턴트가 `growth_stage`, `risk_tags`, `operation_tags`로 stage-aware retrieval을 할 수 있게 만든다.

## 단계별 서브에이전트

### 1. Nursery Agent

- 담당 범위: 발아, 접목 활착, 우량묘 기준, 구입묘 점검
- 우선 소스:
  - `RAG-SRC-001` 고추 재배기술
  - `RAG-SRC-026~027` Grodan Delta 6.5 block
  - `RAG-SRC-038` 모종 구입 및 심기
- 이번에 실제 반영한 핵심 규칙:
  - 구입묘는 작물/품종명이 명확하고, 흰 뿌리가 상토를 단단히 감싼 상태여야 한다.
  - 고추 정식용 구입묘는 보통 `50~72구`, `본엽 11~13매`, `육묘일수 60~80일` 수준을 기준으로 본다.
  - 진딧물, 응애 등 병해충 흔적이 있는 묘는 본포 유입원으로 간주한다.
- overlap 처리:
  - 접목 활착 온습도, 활착실 환경은 `RAG-SRC-001`에 이미 충분한 공식 규칙이 있어 신규 chunk 중복 반영은 하지 않았다.

### 2. Transplanting Agent

- 담당 범위: 정식 시기, 활착 지연, 정식 깊이, 초기 저온/고EC/노화묘 복합 리스크
- 우선 소스:
  - `RAG-SRC-010`, `RAG-SRC-015`, `RAG-SRC-023`
  - `RAG-SRC-039` 노화묘·고EC·깊은 정식 현장 사례
  - `RAG-SRC-026~029` Grodan Delta/GT Master 초기 transfer
- 이번에 실제 반영한 핵심 규칙:
  - 노화묘, 깊은 정식, 고EC 토양이 동시에 겹치면 활착 지연이 길어질 수 있다.
  - 이런 구간은 추가 시비나 강한 생육 촉진보다 먼저 뿌리 회복과 과실 부하 완화가 우선이다.
- overlap 처리:
  - 단순 `노화묘`, 단순 `깊은 정식`은 기존 chunk가 있으므로, 이번에는 `복합 원인` 규칙만 추가했다.

### 3. Vegetative/Fruiting Agent

- 담당 범위: 촉성재배 환경 목표, 개화/착과, 수량 증대, 예방형 병충해 대응
- 우선 소스:
  - `RAG-SRC-011` 촉성/반촉성 일정
  - `RAG-SRC-020` 낙화·웃자람 기술지원
  - `RAG-SRC-028~032` GT Master root zone steering
  - `RAG-SRC-040~042` 촉성재배 작업일정, 적화, 시설해충 예방 보도자료
- 이번에 실제 반영한 핵심 규칙:
  - 촉성재배에서 개화·과비대 적온은 `18~23℃`, 열매달림 가능 온도는 `15.5~28.5℃`다.
  - `접목재배 + 관비재배 + 다겹보온커튼`은 수량/에너지 절감 측면에서 동시에 검토할 가치가 있다.
  - 적화는 `4차분지`까지가 실익이 크고, 후기 착과 집중과 일시 수확에 유리하다.
  - 진딧물, 잎응애, 총채벌레는 밀도가 오른 뒤보다 `5월까지` 예방형 천적 방사가 효과적이다.
- overlap 처리:
  - GT Master의 일반 EC/WC steering은 이미 seed에 반영돼 있어 이번에는 수량·예방 방제와 직접 연결되는 규칙만 추가했다.

### 4. Harvest/Drying Agent

- 담당 범위: 수확 적기, 탄저병 회피 수확, 건조, 저장, 재흡습 방지
- 우선 소스:
  - `RAG-SRC-008`, `RAG-SRC-025`
  - `RAG-SRC-043~044` 수확, 건조 및 저장 농영상
- 이번에 실제 반영한 핵심 규칙:
  - `80% 이상` 붉어진 과실은 즉시 수확해 남은 과실 숙기를 촉진한다.
  - 완전 착색 전 과실은 `2~3일` 그늘 후숙 뒤 건조해야 희나리 위험이 줄어든다.
  - 건조는 햇빛/비닐하우스 자연건조보다 `열풍건조`를 우선하고, 저장은 `함수율 15~17%`에서 차단성 비닐 밀봉, `UV 차단 포장`, `저온 보관`이 기준이다.
- overlap 처리:
  - 기존 `3단계 열풍건조 곡선` chunk는 유지하고, 이번에는 `수확-후숙-저장 연결` 실무 규칙만 보강했다.

## 이번 반영 원칙

- 공식 농업기관 자료를 우선 반영했다.
- 기존 `RAG-SRC-001`, Grodan, 병해충 청크와 중복되는 항목은 제외했다.
- 신규 chunk는 `임계값`, `작업 시점`, `센서/운영 연결`, `실패 예방`이 있는 내용만 남겼다.

## RAG 반영 결과

- 신규 source: `RAG-SRC-038`~`RAG-SRC-044`
- 신규 seed chunk: `8건`
- 반영 후 목표:
  - `growth_stage` 기반 retrieval에서 `nursery`, `transplanting`, `flowering`, `harvest_drying_storage` 구간의 근거 밀도 보강
  - `stage-specific prompt + vector retrieval` 조합에서 stage mismatch 감소
